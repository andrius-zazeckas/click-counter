# Click Counter

This repository contains a basic web application that counts the number of times a button is clicked. The project is built using HTML, CSS, and JavaScript.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/andrius-zazeckas/click-counter.git
2. Navigate to the project directory:
   ```sh
   cd click-counter
3. Open index.html in your favorite web browser to see the responsive webpage in action.
